# 1, Fork this repository.

# 2, Download a decrypted YouTube ipa file from [here](https://armconverter.com/decryptedappstore/us).

# 3, Upload the ipa file to [here](https://litterbox.catbox.moe/) and get the download URL.

# 4, Run workflow under *Build and Release YTLitePlus* at action tab.

# 5, Find the builded ipa file at release page. *eg.https://github.com/fisheepx/YTLitePlus/releases*
